## Overview

SFTP (Secure File Transfer Protocol) provides an encrypted, secure way to transfer import files. This guide walks you through preparing your file, connecting via SFTP, uploading, and checking the import status.

## Supported SFTP applications

- **WinSCP** (recommended)
- FileZilla
- CyberDuck
- FireFTP
- ClassicFTP
- FreeFTP by CoffeeCup

<Steps>
  <Step title="Prepare your CSV file">
    Complete your spreadsheet and save it as a **CSV** file. Ensure all data is correctly formatted -- missing or incorrectly formatted data will cause the import to fail.
  </Step>
  <Step title="Choose an SFTP application">
    Select an SFTP application from the list above. This guide uses **WinSCP** as an example.
  </Step>
  <Step title="Sign into your domain">
    Open your SFTP application, create a new session, and select **SFTP** as the file protocol. Enter the credentials found under **FTP Users** in the web app.
  </Step>
  <Step title="Upload the file">
    Find your CSV file on the left side of the application. Drag it to the right side (the server) to begin the upload.
  </Step>
  <Step title="Check import status">
    Go to **System Logs** in the web app. A **successful** upload shows a confirmation message. A **failed** import shows an error explaining the cause.
  </Step>
</Steps>

<Warning>Always review error messages in System Logs carefully to understand and fix any import issues.</Warning>

<Info>SFTP encrypts file transfers, making it more secure than standard FTP. Always use SFTP to protect your data.</Info>

<AccordionGroup>
  <Accordion title="Where can I find my SFTP credentials?">
    Your credentials are under **FTP Users** in the web app.
  </Accordion>
  <Accordion title="What file format is required?">
    The file must be in **CSV** format.
  </Accordion>
  <Accordion title="What should I do if the import fails?">
    Check the error message in **System Logs**. The issue is usually caused by incorrect formatting or missing information in your CSV file.
  </Accordion>
</AccordionGroup>
