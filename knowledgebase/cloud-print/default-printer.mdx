## Setting your default printer

You can designate one printer as your default so that all print jobs are automatically routed to it. Only one default printer can be active at a time.

<Steps>
  <Step title="Go to the Printers screen">
    From the **Settings** menu, select **Printers**.
  </Step>
  <Step title="Check printer support">
    In the **Printers** list, check whether each printer supports documents and labels. Look for a "Yes" in the relevant columns.
  </Step>
  <Step title="Select a printer">
    Click on the printer you want to set as the default.
  </Step>
  <Step title="Set the printer as default">
    A **General** screen will appear. Tick the appropriate box to set this printer as your default, then click **Save**.
  </Step>
</Steps>

<Info>If no default printer is set, you will need to manually select a printer each time you print.</Info>

<Accordion title="FAQ">
  **Can I set more than one default printer?**

  No, only one printer can be set as the default at a time.

  **How do I change my default printer?**

  Click on the new printer you want to set as default, make your changes on the **General** screen, and click **Save**.

  **Can I change other printer settings on the General screen?**

  Yes, you can adjust various printer settings from the **General** screen before saving.
</Accordion>
