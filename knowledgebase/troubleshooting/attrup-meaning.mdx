ATTRUP stands for **Attribute Update** -- a transaction type that appears when key product details such as batch numbers or sell-by dates are changed within the system.

<Info>
ATTRUP only tracks changes to product attributes (batch numbers, sell-by dates). It does not affect stock levels or trigger general stock adjustments.
</Info>

## When you will see ATTRUP

You will encounter an ATTRUP transaction when:

- A product's **batch number** has been updated
- A product's **sell-by date** has been modified

These changes are logged as ATTRUP transactions in the system's transaction history, making it easy to audit attribute modifications.
