## Overview

The HHD Implementations view has been redesigned to make customisation faster and more visual. Key improvements include a live device simulation, an enhanced flow sequence menu, and the ability to duplicate flows between implementations.

## Real-time handheld simulation

The frontend now features a **simulated handheld view** that provides a live preview of your implementation exactly as it will appear on the physical device. You no longer need to guess how your configurations will look in the field.

## Direct flow duplication

You can now **duplicate a flow from one implementation to another** directly within the frontend, eliminating the need to manually work with back-end JSON files. This simplifies the process of scaling successful configurations across different implementations.
