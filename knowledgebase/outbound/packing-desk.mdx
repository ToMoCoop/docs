## Packing Desk Overview

The packing desk provides a visual representation of the packing process, giving you more clarity and information during order fulfilment.

### 1. Accessing the Packing Desk

Navigate to the **Goods Out** tab in the main menu and select **Packing Desk**.

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-1.png" alt="Goods Out menu showing Packing Desk option" />
</Frame>

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-2.png" alt="Main Packing Desk screen" />
</Frame>

The main Packing Desk screen displays all current pack tasks associated with a pick list and provides these key functionalities:

- **View current pack tasks** -- See all pack tasks awaiting processing.
- **Assign a printer** -- Ensure labels and documents print to the correct device.
- **Search for pack tasks** -- Locate orders using Storage Unit Reference, Sales Order Reference, or Pick List Reference. These can also be scanned using a scanner or HHD.

---

### 2. Preparing to Pack

Before packing, ensure the following requirements are met:

- **Dispatch Strategy**: Must be set to **Shipment**.
- **Pack Strategy**: Must be set (either Pack by Line or Pack by Unit).
- **Complete Delivery Address**: Must include address lines, postcode, phone number, and email.
- **Pick List Status**: All pick tasks must be completed to generate pack tasks.

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-3.png" alt="Pack task requirements" />
</Frame>

---

### 3. Starting a Pack

<Steps>
  <Step title="Locate the pack">
    Use the search fields to find the order you want to pack.
  </Step>
  <Step title="Begin the packing process">
    Either scroll right and click **Begin Pack**, or **scan the reference** to start packing.

    <Frame>
      <img src="/images/knowledgebase/packing-desk-user-guide-4.png" alt="Begin Pack button" />
    </Frame>

    A menu will appear with available **Storage Unit Types**. These can be managed via **Settings** > **Storage Unit Types**.

    <Frame>
      <img src="/images/knowledgebase/packing-desk-user-guide-5.png" alt="Storage Unit Types selection" />
    </Frame>

    Select the storage unit type you wish to pack into.
  </Step>
</Steps>

---

### 4. Packing Items

Once you select a storage unit type, you will see the **Packing Screen**:

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-6.png" alt="Packing screen layout" />
</Frame>

- **Left Panel**: Displays the sales order reference, delivery address, packing notes, and products with quantities.
- **Right Panel (Packing Summary)**: Shows the package contents, dimensions, and total weight. You can also change the storage unit type here.

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-7.png" alt="Packing summary panel" />
</Frame>

#### Moving Items into the Package

Scan items to move them into the packing summary, or use the arrows next to items to manually move them across. Repeat until all items are packed.

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-8.png" alt="Items being packed" />
</Frame>

---

### 5. Finalising the Package

Once all items are packed:

1. Click **Close Package and Print Labels**. This completes the packing process, submits the shipment, and prints labels automatically.
2. Confirm the package is reflected as shipped in the **Sales Order / Pick List** screen.

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-9.png" alt="Close Package button" />
</Frame>

<Frame>
  <img src="/images/knowledgebase/packing-desk-user-guide-10.png" alt="Shipment confirmed status" />
</Frame>

### Key Takeaways

- Ensure all order details and pick tasks are complete **before starting** packing.
- Use the **Packing Summary** to verify weights, dimensions, and contents.
- **Close Package and Print Labels** finalises the shipment and triggers label printing.
