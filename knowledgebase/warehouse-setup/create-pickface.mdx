A **pickface** is a designated location used for picking smaller quantities of stock close to where they're needed. Configuring a location as a pickface and assigning it to a product lets Clarus manage stock availability automatically through one of two replenishment strategies.

## How setup fits together

There are three steps to getting a pickface working:

1. **Enable a location** as a pickface.
2. **Assign the pickface to a product** — either by selecting specific locations or by using a location group.
3. **Choose a replenishment strategy** — Continuous or Demand — for each assigned location.

## Set up a pickface

<Steps>
<Step title="Enable the location as a pickface">
Go to the **Locations** screen and select the location you want to use. Open the menu for that location and set **Is Pickface** to **Yes**. The location can now be assigned to a product.

<Info>
If you plan to assign a [location group](/knowledgebase/warehouse-setup/location-groups) as a pickface, every location in the group must have **Is Pickface** set to **Yes**.
</Info>
</Step>
<Step title="Assign the pickface to a product">
Navigate to the **Products** screen and find the product. Open the **Pickface** panel and click **New** to add a location, or assign a **Location Group** to manage multiple pickfaces at once.

Use the **arrow controls** on the left of each row to set the order Clarus uses when distributing picks and replenishments.
</Step>
<Step title="Choose a replenishment strategy">
For each assigned location, choose **Continuous** or **Demand** replenishment. The strategy decides how and when Clarus tops the location up. See [Pickface Replenishment](/knowledgebase/warehouse-setup/replenishment/overview) for guidance on choosing between them, and follow the strategy-specific setup pages for the remaining configuration.
</Step>
</Steps>
