## Overview

Standard users have wide-ranging access, allowing them to view and manage most system features. This role is suited for regular office and warehouse staff who need broad operational access without full administrative control.

## Permissions summary

| Area | View | Manage |
|------|------|--------|
| Accounts | Yes | Yes |
| Alternative codes | Yes | Yes |
| Carriers | Yes | Yes |
| Charges | Yes | Yes |
| Addresses | Yes | Yes |
| Goods In | Yes | Yes |
| Goods Out | Yes | Yes |
| Pick lists | Yes | Yes |
| Loads | Yes | Yes |
| Locations | Yes | Yes |
| Printers | Yes | No |
| Products | Yes | Yes |
| Reports | Yes | Yes |
| Stock | Yes | Hold, release, move, adjust |
| Tasks | Yes | Yes |
| Transactions | Yes | No |
| Warehouses | Yes | Yes |

<Info>Standard users have access to warehouses according to their assigned access, which an admin can customise.</Info>

<AccordionGroup>
  <Accordion title="Are there features Standard users cannot manage?">
    Yes. Standard users have limited access to certain system settings and administrative functions reserved for Admin users.
  </Accordion>
  <Accordion title="Can Standard users run reports?">
    Yes. Standard users can both view and manage reports.
  </Accordion>
</AccordionGroup>
