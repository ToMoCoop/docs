## Overview

You can hide specific usernames from other users within the system. This ensures that clients interact only with the profiles relevant to them, maintaining a secure and organised workspace.

## Setting up restricted visibility

<Steps>
  <Step title="Access Settings">
    From the home dashboard, navigate to the **Settings** menu.
  </Step>
  <Step title="Manage Users">
    Within Settings, select the **Users** tab.
  </Step>
  <Step title="Select a profile">
    Choose an existing user to edit, or select **Create New User**.
  </Step>
  <Step title="Adjust visibility toggles">
    You will see options to view all accounts, warehouses, and users. Switch the **"View All Users"** toggle to **No**.

    <Frame><img src="/images/knowledgebase/enhanced-privacy-restrict-user-1.png" alt="User visibility toggle showing View All Users set to No" /></Frame>
  </Step>
  <Step title="Define approved users">
    Once disabled, a new **Approved Users** tab appears on the user's profile page.
  </Step>
  <Step title="Add specific permissions">
    Click the selection box at the bottom of the page to see a list of usernames. You can type to filter the list. Select the desired username and click **Assign**.

    <Frame><img src="/images/knowledgebase/enhanced-privacy-restrict-user-2.png" alt="Approved Users tab with username selection and Assign button" /></Frame>
  </Step>
</Steps>

<Info>Repeat this process for every individual you want the user to be able to see. Once saved, their visibility will be strictly limited to your approved list.</Info>
