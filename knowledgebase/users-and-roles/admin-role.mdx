## Overview

The Admin role provides **complete control** over all features and settings. Admins can manage accounts, carriers, products, warehouses, users, and system configurations.

## Admin capabilities

| Area | Access |
|------|--------|
| Users and roles | Create, edit, delete users and assign roles |
| Accounts and warehouses | Full access to all registered accounts and warehouses |
| System settings | Edit and configure all settings |
| Reports, products, carriers | View and modify all data |
| Stock | Full control including adjustments |

<Info>There are no features that Admins cannot access. The Admin role has unrestricted access to all functions within the system.</Info>

<AccordionGroup>
  <Accordion title="Can Admins restrict their own access?">
    No. Admins cannot restrict their own access, but they can manage the access of other users.
  </Accordion>
  <Accordion title="Can Admins change user roles and access levels?">
    Yes. Admins can create, edit, and delete users and assign different roles and permissions.
  </Accordion>
</AccordionGroup>
