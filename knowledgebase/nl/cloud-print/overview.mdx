Met Cloud Print kun je documenten en labels rechtstreeks vanuit Clarus WMS afdrukken naar printers op je lokale netwerk — zonder complexe VPN-instellingen, port forwarding of directe netwerktoegang vanuit de cloud.

Een lichte **Cloud Print-agent** draait op een machine in je netwerk en maakt een uitgaande, versleutelde verbinding met Clarus. Afdruktaken worden via die verbinding verzonden naar elke printer waar de hostmachine al naar kan afdrukken.

## Waarom Cloud Print gebruiken

- **Geen inkomende firewallregels.** De agent maakt alleen uitgaande verbindingen, dus je hoeft geen poorten te openen of printers bloot te stellen aan het internet.
- **Werkt door bedrijfsfirewalls en NAT.** Geen VPN of port forwarding nodig.
- **Automatische printerdetectie.** De agent vindt alle printers die beschikbaar zijn op de hostmachine — USB, netwerk, alles met geïnstalleerde stuurprogramma's.
- **Multi-platform.** Werkt op Windows, macOS en Linux.

## Beveiliging

- **Versleuteld tijdens verzending.** Alle communicatie gebruikt TLS 1.2+.
- **Geen gegevensretentie.** Afdruktaken worden permanent uit de cloud verwijderd zodra het afdrukken is voltooid.
- **Ondertekende binaries.** Het agent-installatieprogramma is digitaal ondertekend.
- **Alleen uitgaand.** Geen inkomende netwerktoegang vereist, waardoor het aanvalsoppervlak minimaal blijft.
