## Hoe te installeren en te gebruiken Clarus WMS Cloud Print

In deze handleiding wordt uitgelegd hoe u Clarus WMS Cloud Print installeert en integreert met uw systeem. Volg deze stappen om Cloud Print in te stellen en deze te gebruiken voor het rechtstreeks vanuit Clarus WMS afdrukken.

## Stapsgewijze handleiding

### 1. Instellingen openen

Klik in het menu aan de linkerkant van uw Clarus WMS-scherm op **Instellingen** . Selecteer vervolgens **Integraties** Om alle huidige integraties te zien die beschikbaar zijn voor uw systeem.

### 2. Een nieuwe integratie toevoegen

Klik op de **+ Integratie toevoegen** knop. Een lijst met beschikbare integraties verschijnt aan de rechterkant van uw scherm.

### 3. Selecteer Clarus WMS Cloud Print

Zoek **Clarus WMS Cloud Print** Klik in de lijst en klik op de **+ toevoegen** knop.

### 4. Noem de integratie

Voer een naam in voor uw nieuwe integratie (bijvoorbeeld ""Cloud Print Setup""). Klik vervolgens **Opslaan** .

### 5. Download Cloud Print

Na het sparen wordt u naar een **Integratie bewerken** scherm. Dit scherm geeft instructies voor het downloaden van Cloud Print. Kies uw besturingssysteem (ofwel **Microsoft Windows** Of **Mac** ).

### 6. Let op uw gebruikersnaam en wachtwoord

Op de **Integratie bewerken** Als u op het scherm ontvangt u een gebruikersnaam en wachtwoord. Houd deze gegevens bij de hand, want u hebt ze nodig tijdens het installatieproces.

[CALLOUT:WARNING] **Herinnering:** U hebt deze gebruikersnaam en wachtwoord nodig om later in te loggen op Cloud Print.

### 7. Cloud Print installeren

Nadat u het installatiebestand hebt gedownload, opent u het bestand en selecteert u **Rennen** . Volg de installatie-instructies op het scherm, vergelijkbaar met andere softwareinstallaties.

### 8. Inloggen op Cloud Print

Na de installatie wordt u gevraagd aan te melden met de gebruikersnaam en wachtwoord die eerder zijn opgegeven.

### 9. Maak de set-up compleet

Eenmaal ingelogd, verschijnt de integratie op uw **Integraties** scherm, dat bevestigt dat Clarus WMS Cloud Print succesvol is ingesteld.

## Multi-user omgevingen

Voor servers of machines met meerdere ingelogde gebruikers — zoals terminal servers of gedeelde werkstations — installeert u de agent als een **Windows Service** in plaats van als een desktopapplicatie. Dit zorgt ervoor dat:

- De agent draait ongeacht welke gebruiker is ingelogd
- Alle gebruikers op de machine consistent toegang hebben tot dezelfde printers
- De agent automatisch start wanneer de machine opstart

<Tip>Installeer de Cloud Print-agent met een account dat al toegang heeft tot elke printer die u nodig heeft. De agent detecteert printers met behulp van de weergave van dat account, dus elke printer die het niet kan zien op het moment van installatie verschijnt niet in Clarus WMS.</Tip>

## Systeemvereisten

### Ondersteunde besturingssystemen

| Platform | Versies                                                     |
| -------- | ----------------------------------------------------------- |
| Windows  | Windows 10, Windows 11, Windows Server 2016 en later        |
| macOS    | macOS 10.14 (Mojave) en later                               |
| Linux    | Ubuntu, Debian, CentOS, RHEL, Raspbian en anderen           |

### Hardware

| Resource     | Minimum                              | Aanbevolen                              |
| ------------ | ------------------------------------ | --------------------------------------- |
| CPU          | 1 kern (elke moderne processor)      | 2+ kernen                               |
| Geheugen     | 512 MB beschikbaar RAM               | 1 GB beschikbaar RAM                    |
| Schijfruimte | 100 MB                               | 500 MB (voor buffering van afdruktaken) |

<Note>De agent is licht. Elke machine die een moderne webbrowser kan draaien, kan de Cloud Print-agent comfortabel draaien.</Note>

### Netwerk

| Vereiste            | Details                                                     |
| ------------------- | ----------------------------------------------------------- |
| Internetverbinding  | Stabiele breedband                                          |
| Uitgaande poorten   | 443 (HTTPS), 6123 (WebSocket)                               |
| DNS-resolutie       | Moet de Cloud Print service-endpoints kunnen oplossen       |

Voor de volledige endpoint-lijst en de architectuur achter de agent-verbinding, zie [Hoe Cloud Print werkt](/knowledgebase/nl/cloud-print/how-it-works).

## FAQs

### Waar vind ik de integratie van Cloud Print?

Je kunt het vinden in de **Instellingen** sectie onder **Integraties** bij Clarus WMS.

### Wat moet ik doen als ik mijn gebruikersnaam en wachtwoord verlies?

U kunt uw gebruikersnaam en wachtwoord ophalen bij de **Integratie bewerken** Het scherm van Clarus WMS.

### Kan ik Cloud Print installeren op zowel Windows als Mac?

Ja, Clarus WMS Cloud Print ondersteunt zowel Microsoft Windows- als Mac-besturingssystemen.

### Wat gebeurt er nadat ik Cloud Print heb geïnstalleerd?

Na de installatie moet u inloggen met uw gebruikersnaam en wachtwoord. De integratie verschijnt dan in uw lijst met opgeslagen integraties.

### Kan ik de Cloud Print integratie later verwijderen?

Ja, u kunt de integratie beheren of verwijderen uit de **Integraties** Het menu in Clarus WMS op elk gewenst moment.
