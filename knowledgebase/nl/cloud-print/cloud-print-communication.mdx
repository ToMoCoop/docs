## Printertoegang voor Clarus WMS Cloud Print

In deze handleiding wordt uitgelegd hoe u ervoor kunt zorgen dat uw printers zichtbaar zijn voor de Clarus WMS Cloud Print-client en hoe communicatie met deze printers werkt. Als u deze stappen volgt, kunt u uw afdrukomgeving correct instellen.

## Stapsgewijze handleiding

### 1. Zorgen voor de printerzichtbaarheid

De Clarus WMS Cloud Print-client moet zijn geïnstalleerd op een computer waar de printers die u wilt gebruiken al lokaal zijn geïnstalleerd. Dit betekent dat ze moeten worden weergegeven in het besturingssysteem "Drukkers en scanners" van uw computer.

### 2. Printercommunicatie begrijpen

Zodra de Clarus WMS Cloud Print-client is geïnstalleerd, communiceert deze rechtstreeks met uw printers. Op Windows gebruikt de client bijvoorbeeld de Windows API om met de Windows-service afdrukspooler Windows Service te praten.

## FAQs

### Wat betekent 'lokaal geïnstalleerd' voor printers?

Dit betekent dat de printers worden ingesteld en herkend door de computer waar de Clarus WMS Cloud Print-client is geïnstalleerd. U zou ze moeten kunnen zien in de instellingen 'Drukkers en scanners' van uw computer.

### Hoe communiceert de Clarus WMS Cloud Print client met printers op Windows?

Op Windows communiceert de Clarus WMS Cloud Print-client rechtstreeks met de Windows Service Print Spooler Windows Service met behulp van de Windows API. Dit zorgt voor efficiënt en betrouwbaar printen.
