MFA instellen duurt ongeveer twee minuten. Nadat je op **Set up MFA** hebt geklikt in de [aanmeldmelding](/knowledgebase/nl/security/mfa-login-prompt), toont Clarus WMS het scherm **Set Up Authenticator**. Je scant de QR-code met een authenticator-app op je telefoon en voert vervolgens een verificatiecode in om te bevestigen.

<MfaRollout />

## Voordat je begint

Je hebt nodig:

- Je telefoon of tablet, ontgrendeld en binnen handbereik
- Een authenticator-app geïnstalleerd op dat apparaat

Populaire authenticator-apps:

- **1Password** — als je team het al gebruikt als wachtwoordmanager, werkt het ook als authenticator
- **Authy** — gratis, ondersteunt back-ups op meerdere apparaten
- **Google Authenticator** — gratis, eenvoudig
- **Microsoft Authenticator** — gratis, ondersteunt back-ups via je Microsoft-account

<Tip>Elke TOTP-compatibele authenticator-app werkt. Kies de app die je organisatie al gebruikt als er een voorkeur is.</Tip>

## Hoe je de instelling start

De meeste mensen bereiken de instelling via de aanmeldmelding — zodra MFA beschikbaar is op je account, zie je deze de volgende keer dat je inlogt. Klik op **Set up MFA** om naar het scherm **Set Up Authenticator** te gaan.

Als je eerder **Remind me later** of **I don't want MFA** hebt gekozen, kun je het instellen ook starten vanuit [MFA inschakelen in Security Settings](/knowledgebase/nl/security/enable-mfa). Beide routes leiden naar hetzelfde scherm.

## MFA instellen

<Steps>
  <Step title="Open het scherm Set Up Authenticator">
    Klik op **Set up MFA** in de aanmeldmelding. Het scherm **Set Up Authenticator** verschijnt met een QR-code en een handmatige instelcode.

    <Frame>
      <img src="/images/knowledgebase/set-up-mfa-1.png" alt="Het scherm Set Up Authenticator met een QR-code, een handmatige instelcode eronder en een veld Verification Code met de knop Verify Code." />
    </Frame>
  </Step>
  <Step title="Open je authenticator-app">
    Open op je telefoon je authenticator-app en kies **een nieuw account toevoegen** (de exacte bewoording is afhankelijk van de app — zoek naar een **+**-knop of **Add Account**).
  </Step>
  <Step title="Scan de QR-code">
    Gebruik de scanner van de app om de camera van je telefoon op de QR-code op het scherm te richten. De app voegt een nieuwe vermelding toe voor Clarus WMS en begint met het genereren van codes van zes cijfers die elke 30 seconden vernieuwen.

    <Tip>Als je telefoon de QR-code niet kan scannen, gebruik dan de handmatige instelcode die eronder wordt weergegeven. Kies in je authenticator-app om een account toe te voegen door een **instelsleutel in te voeren** (de bewoording verschilt per app) en plak vervolgens de code van het Clarus WMS-scherm.</Tip>
  </Step>
  <Step title="Voer de verificatiecode in">
    Typ in Clarus WMS de **zescijferige code** die momenteel in je authenticator-app wordt weergegeven in het veld **Verification Code** en klik op **Verify Code**.
  </Step>
  <Step title="Klaar">
    MFA is nu actief op je account. Vanaf de volgende keer inloggen wordt je na het invoeren van je wachtwoord om een code uit je authenticator-app gevraagd.
  </Step>
</Steps>

<Warning>Codes vernieuwen elke 30 seconden. Als een code wordt geweigerd, wacht dan op de volgende en probeer het opnieuw — je hebt de code mogelijk net na het verlopen ingevoerd.</Warning>

## De volgende keer dat je inlogt

Na je wachtwoord verschijnt het scherm **Two-Factor Authentication**. Open je authenticator-app, zoek de vermelding **Clarus WMS** en typ de huidige zescijferige code in het veld **Verification Code** en klik op **Verify Code**. De code is ongeveer 30 seconden geldig voordat deze vernieuwt.

<Frame>
  <img src="/images/knowledgebase/set-up-mfa-2.png" alt="Het Two-Factor Authentication-aanmeldscherm met een veld Verification Code en een knop Verify Code." />
</Frame>

<AccordionGroup>
  <Accordion title="Moet ik dit op elk apparaat doen waarvan ik inlog?">
    Nee. MFA is gekoppeld aan je account, niet aan je apparaat. Je wordt om een code uit je authenticator-app gevraagd wanneer je inlogt, ongeacht welke browser of welk apparaat je gebruikt.
  </Accordion>
  <Accordion title="Kan ik meer dan één authenticator-app of apparaat gebruiken?">
    Nee. Clarus WMS ondersteunt één authenticator-vermelding per gebruiker. Als je wilt overstappen naar een andere app of een nieuwe telefoon, schakel MFA dan uit vanuit je Security Settings en stel het opnieuw in op het nieuwe apparaat — zie [MFA uitschakelen in Security Settings](/knowledgebase/nl/security/disable-mfa) en vervolgens [MFA inschakelen in Security Settings](/knowledgebase/nl/security/enable-mfa).
  </Accordion>
  <Accordion title="Wat als ik mijn telefoon verlies?">
    Zie [MFA-problemen](/knowledgebase/nl/troubleshooting/mfa-issues) voor hersteloptie.
  </Accordion>
</AccordionGroup>
