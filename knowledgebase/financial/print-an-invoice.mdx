## Printing Invoices

You have two methods for printing invoices: saving as a PDF for a digital record, or direct physical printing using Cloud Print.

### Save to PDF

Save the invoice as a PDF file. This is useful for keeping a digital record or sharing it via email. Once saved, you can print the PDF using any standard PDF viewer.

### Use Cloud Print

If you need a printed copy, you can use **Cloud Print** to print the invoice directly from the system without downloading the file first.

<Tip>For printing many invoices together, download them as PDFs first. Cloud Print typically handles one invoice at a time.</Tip>

<Info>If your printer is not connected to Cloud Print, check your printer settings for compatibility. Contact support if problems continue.</Info>
