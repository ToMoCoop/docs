## Exporting Supplier Information

You can export your supplier data to an Excel file for easy viewing and reporting.

<Steps>
  <Step title="Click the Export button">
    At the top of the supplier screen, click the **Export** button.
  </Step>
  <Step title="Download the Excel file">
    An **Excel document** with all your supplier data will be ready for download. The file saves to your browser's default download location (usually the **Downloads** folder).
  </Step>
</Steps>

<Tip>You can customise which columns are visible before exporting. The exported file will include all currently visible columns. Use filters to export only active suppliers if needed.</Tip>
