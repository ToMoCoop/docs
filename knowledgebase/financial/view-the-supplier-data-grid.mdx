## Managing and Viewing Suppliers

The supplier data grid lets you view and manage all your suppliers in one place.

<Steps>
  <Step title="Access the supplier screen">
    Navigate to **Suppliers** from the left-hand menu. This displays all suppliers in the system.
  </Step>
  <Step title="Understand default columns">
    By default, you will see the following columns:

    - **ID**
    - **Code**
    - **Name**
    - **Description**
    - **Active?**
  </Step>
  <Step title="Customise columns">
    Adjust the data view by adding or removing columns using the column manager. To sort columns alphabetically, click on the column heading you wish to sort by.
  </Step>
</Steps>

<Tip>Click on any column heading to sort the supplier list by that field. Use the column manager to show or hide columns as needed.</Tip>
