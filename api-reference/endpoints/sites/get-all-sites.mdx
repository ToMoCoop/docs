---
title: "Get All Sites"
description: "Use the GraphQL API to retrieve all sites."
openapi: "POST /graphql"
---

<RequestExample>
```http Query
query getSites(
    $first: Int,
    $after: String
) {
    sites{
        all(
            after: $after
            first: $first
        ) {
            pageInfo {
                hasNextPage
                hasPreviousPage
                startCursor
                endCursor
            }
            edges {
                cursor
                node {
                    id
                    code
                    name
                    boolean1
                    boolean2
                    boolean3
                    boolean4
                    boolean4
                    boolean5
                    datetime1
                    datetime1_formatted
                    datetime2
                    datetime2_formatted
                    datetime3
                    datetime3_formatted
                    datetime4
                    datetime4_formatted
                    datetime5
                    datetime5_formatted
                    default
                    external_system_reference1
                    external_system_reference2
                    external_system_reference3
                    integer1
                    integer2
                    integer3
                    integer4
                    integer5
                    string1
                    string2
                    string3
                    string4
                    string5
                    tier
                    created_at
                    created_at_formatted
                    updated_at
                    updated_at_formatted
                }
            }
        }
    }
}
```
</RequestExample>
